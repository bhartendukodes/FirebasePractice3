# Firebase Practice

A hands-on Firebase practice project in Kotlin, exploring various Firebase services including Authentication, Firestore, and Cloud Storage.

## Features

- Firebase Authentication (Email, Google)
- Cloud Firestore CRUD operations
- Firebase Cloud Storage
- Real-time data sync

## Tech Stack

- **Language:** Kotlin
- **Backend:** Firebase (Auth, Firestore, Storage)
- **UI:** Material Design

## Getting Started

1. Clone the repository
```
git clone https://github.com/bhartendukodes/FirebasePractice3.git
```
2. Open in Android Studio
3. Add your `google-services.json`
4. Build and run
